API do Corinthians

Uma API que fornece informações sobre o elenco do Corinthians, incluindo elenco atual, elenco de 2024, e divisão por posições.

Principais Funcionalidades:

  • Elenco Atual: Listagem completa dos jogadores atuais.
  • Elenco 2024: Numeros do elenco em 2024
  • Posições: Dados organizados por:
    • Atacantes
    • Meias
    • Defensores
    • Goleiros
