Chatbot FURIA

Este projeto é um chatbot interativo desenvolvido para a FURIA, focado em fornecer informações atualizadas sobre a equipe de Counter-Strike 2 (CS2). A aplicação foi construída utilizando Python com Flask no backend, permitindo uma integração eficaz com a API Cohere para gerar respostas automáticas com base nas perguntas dos usuários. O armazenamento de dados, como informações sobre jogadores e partidas, é feito de forma eficiente no MongoDB.


No frontend, a interface foi construída utilizando Angular, proporcionando uma navegação fluida e responsiva. O design foi projetado para ser simples e eficiente, com o objetivo de oferecer uma experiência agradável ao usuário em qualquer dispositivo. A comunicação entre o frontend e o backend é feita por meio de uma API RESTful, garantindo uma interação rápida e precisa.


A aplicação também inclui um sistema de histórico de conversa, permitindo que o usuário veja as últimas interações com o chatbot. Para garantir uma experiência personalizada, o chatbot é capaz de fornecer informações detalhadas sobre os jogadores da FURIA, próximos jogos, resultados recentes e muito mais.


O projeto segue a arquitetura MVC, o que facilita a manutenção e a escalabilidade do código. Com versionamento utilizando Git e hospedagem no GitHub, o código está organizado e pronto para futuras melhorias.

Funcionalidades Principais

  • Frontend: Interface dinâmica e responsiva desenvolvida com Angular.
  • Backend: Implementado com Python e Flask, com integração à API Cohere para respostas automáticas.
  • Armazenamento: MongoDB para gerenciar informações sobre jogadores e partidas.
  • Histórico de Conversa: Sistema de armazenamento de mensagens para consultas futuras.
  • Informações Personalizadas: Respostas detalhadas sobre jogadores, próximos jogos e resultados recentes.
