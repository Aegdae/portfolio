Rede Social

Este projeto é uma rede social completa, projetada para oferecer uma experiência interativa e intuitiva aos usuários. A aplicação foi construída utilizando Flask no backend, com o objetivo de garantir uma autenticação segura, gerenciamento de dados eficientes e uma estrutura robusta para o funcionamento das funcionalidades. A interação com o banco de dados é feita por meio do PostgreSQL, garantindo integridade e performance.

No frontend, a interface foi cuidadosamente projetada utilizando HTML, CSS e JavaScript, criando uma plataforma moderna e fácil de usar. A navegação intuitiva permite que os usuários criem e gerenciem seus perfis, interajam com postagens, curtam e comentem conteúdos, e se conectem de forma simples e eficaz.

Além disso, a aplicação inclui funcionalidades de login seguro com bcrypt, proporcionando uma camada extra de segurança na autenticação dos usuários. O design é totalmente responsivo, permitindo uma experiência fluida em dispositivos móveis, tablets e desktops.

O projeto segue o padrão de arquitetura MVC para organizar e estruturar o código, o que facilita a manutenção e o desenvolvimento de novas funcionalidades. Com um sistema de controle de versão usando Git e hospedagem no GitHub, o código está bem estruturado e pronto para ser escalado ou adaptado conforme as necessidades.

Funcionalidades Principais

  • Frontend: Interface dinâmica e responsiva com HTML, CSS e JavaScript.
  • Backend: Implementado com Flask e PostgreSQL, com foco em desempenho e segurança.
    • Autenticação: Login seguro utilizando bcrypt.
    • Recuperação de Senha: Funcionalidade de recuperação de senha via e-mail.
    • Perfis de Usuário: Cada usuário pode criar e editar seu perfil.
    • Sistema de Postagens: Permite a criação de postagens públicas ou privadas, com interação entre os usuários.
