Rede Social

Este projeto é uma rede social completa, desenvolvida para proporcionar uma experiência dinâmica e interativa. No backend, utilizamos Flask para gerenciar a autenticação, banco de dados e toda a lógica da aplicação. Já no frontend, HTML, CSS e JavaScript foram combinados para criar uma interface moderna e intuitiva. Entre as funcionalidades, estão o registro e login de usuários, criação e interação com postagens, sistema de curtidas e um design responsivo para diferentes dispositivos

Funcionalidades Principais

  • Frontend: Interface dinâmica e responsiva com HTML, CSS e JavaScript.
  • Backend: Desenvolvido em Flask com PostgreSQL.
    • Autenticação segura com bcrypt.
    • Recuperação de senha via e-mail.
    • Perfil de usuário com opção de edição.
    • Sistema de postagens públicas ou privadas.
