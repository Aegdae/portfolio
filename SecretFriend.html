API de Amigo Secreto

Esta aplicação web facilita a organização de amigo secreto, automatizando a criação de grupos e sorteios. O backend em Flask gerencia a lógica e os dados, enquanto o frontend em HTML, CSS e JavaScript oferece uma interface simples e intuitiva. Os usuários podem criar grupos, adicionar participantes e realizar sorteios de forma rápida e segura.

Principais Funcionalidades:

Grupos:
  • Criar, listar e excluir grupos.
Participantes:
  • Adicionar, listar e remover participantes de grupos.
Sorteio:
  • Realizar sorteios automáticos de amigo secreto.
