API de Amigo Secreto

Uma aplicação web para gerenciar grupos de amigo secreto e realizar sorteios de forma automática. Com backend em Flask e frontend em HTML, CSS e JavaScript, a API permite criar grupos, adicionar participantes e realizar sorteios.

Principais Funcionalidades:

Grupos:
  • Criar, listar e excluir grupos.
Participantes:
  • Adicionar, listar e remover participantes de grupos.
Sorteio:
  • Realizar sorteios automáticos de amigo secreto.
